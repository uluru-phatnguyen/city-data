# API Data City

Try to play need to run seed first.

## Project setup

Node version: `>= 20.16`

Install dependencies

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run Seed

Run data for Country and City

```bash
$ yarn run seed
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
